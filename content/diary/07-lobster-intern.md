## 📅 2026-03-20 日记

今天想聊一个有点"偏门"但特别重要的话题：**如何正确"教"龙虾（OpenClaw）。**

不是教你怎么写 Prompt，而是教你怎么**和龙虾相处**。

这个灵感来自两件事：

1. 上周帮朋友调试 OpenClaw 指令，发现他有个习惯：AI 输出不满意，他直接自己重写
2. 看到一篇养龙虾的科普，说很多人养龙虾养不活，是因为"太勤快"

这两件事放在一起，我突然意识到：

**养龙虾（OpenClaw）、带实习生、教 AI，其实是同一件事。**

---

## 🤔 一、为什么是"养龙虾"和"带实习生"？

### 养龙虾的误区

很多人第一次养龙虾，会犯一个错误：**太勤快。**

- 龙虾不动，觉得它饿了，喂
- 水有点浑，觉得它脏，换
- 它躲起来，觉得它不舒服，捞

结果呢？龙虾被折腾死了。

**正确的养法是：** 给它合适的环境，然后**看着它自己活。**

### 带实习生的误区

带过实习生的朋友都有体会：

- 他做得慢，你忍不住接手："算了算了，我来"
- 他做得不好，你直接改完："这样不对，应该..."
- 他问问题，你给完整答案："就这样这样这样"

结果呢？实习生三个月，啥也没学会。

**正确的带法是：** 给他清晰的任务，然后**看着他改好。**

### 教 AI 的误区

是不是特别熟悉？

- AI 输出不理想，你直接自己重写
- AI 理解错了，你换个人工智能
- AI 答得不好，你觉得"AI 也就这样了"

**正确的教法是：** 给它清晰的指令，然后**陪着它迭代。**

## ⚠️ 三、常见错误：你是在"养"还是在"替"？

### 错误 1：AI 输出不好，直接放弃

❌ **典型心态：**
```
"这个 AI 太笨了，还是我自己写吧。"
"试了三次都不对，算了算了。"
"AI 写的东西没法用，还得我重写。"
```

✅ **正确心态：**
```
"它没理解我的意思，我换个说法。"
"第一次输出是素材，不是成品。"
"我教得不够清楚，不是它笨。"
```

**记住：AI 不是魔法，是工具。**

工具用不好，先别怪工具，看看自己会不会用。

---

### 错误 2：期望一次完美

❌ **典型心态：**
```
"我都说了这么多，它应该一次就懂吧？"
"怎么还要我改指令？这不智能啊。"
"别人用 AI 一次成功，我怎么就不行？"
```

✅ **正确心态：**
```
"第一次是草稿，第二次是修改稿，第三次才是成品。"
"迭代是常态，一次完美是例外。"
"别人的一次成功，背后可能试了 10 次。"
```

**记住：好指令是调出来的，不是写出来的。**

就像你教新人，说一遍他就会了？那是天才，不是普通人。

---

### 错误 3：不总结不沉淀

❌ **典型做法：**
```
- 每次用 AI 都从零开始写指令
- 同样的问题问 10 遍，每次都重新描述
- 好的指令不用了，下次重新写
```

✅ **正确做法：**
```
- 建立自己的指令模板库
- 同样的问题，用同样的指令
- 好的指令保存下来，反复用
```

**记住：不沉淀，永远在原地踏步。**

就像你带实习生，教一遍他记住了，下次就不用教了。

AI 也一样。

## 🎯 四、正确做法：像带实习生一样带 AI

### 第 1 步：给清晰的任务（别模糊）

❌ **模糊指令：**
```
帮我写个文案
```

✅ **清晰指令：**
```
你是一个资深文案，帮我写一个小红书风格的咖啡店推广文案，
目标受众是 25-35 岁白领女性，语气轻松活泼，300 字以内。
```

**就像你给实习生派活：**

❌ "你去弄个 PPT"
✅ "你做一个产品介绍 PPT，10 页左右，风格简洁商务，明天下午给我"

看到区别了吗？

---

### 第 2 步：给示例（别光说）

❌ **光说要求：**
```
写得专业一点
```

✅ **给示例：**
```
参考这种风格：
"姐妹们！今天说个扎心的真相..."
```

**就像你教实习生：**

❌ "写得专业点"
✅ "参考我上周写的那个方案，就那种风格"

看到区别了吗？

---

### 第 3 步：给反馈（别只说"不行"）

❌ **模糊反馈：**
```
这个不行，重写
```

✅ **具体反馈：**
```
开头不够吸引人，换成"姐妹们！今天说个扎心的真相..."
正文分 3 段，现在太长了
结尾加个互动引导，让读者评论
```

**就像你改实习生的作业：**

❌ "这不行，重写"
✅ "开头太啰嗦，直接说重点；中间分 3 点，现在太乱了；结尾加个总结"

看到区别了吗？

---

### 第 4 步：给时间（别催）

❌ **急躁做法：**
```
"怎么还没好？我都等 5 分钟了！"
"算了算了，我自己来快。"
```

✅ **正确做法：**
```
"你慢慢写，写好了给我看。"
"第一次写不好正常，改一改就好了。"
```

**就像你带实习生：**

❌ "你怎么这么慢？我来！"
✅ "你慢慢做，做好了给我看，我帮你改"

看到区别了吗？

---

### 第 5 步：给鼓励（别只批评）

❌ **只批评：**
```
"这写的什么玩意？"
"还不如我自己写。"
```

✅ **鼓励 + 建议：**
```
"这次比上次好多了！开头很吸引人。
就是中间部分有点乱，再调整一下结构就更好了。"
```

**就像你带实习生：**

❌ "这写的什么？"
✅ "这次进步很大！这个部分写得特别好，那个部分再改改就更好了"

看到区别了吗？

## 📊 五、迭代过程对比

### ❌ 错误方式（3 次放弃）

```
第 1 次：AI 输出 60 分
你："这什么玩意？"
自己重写

第 2 次：AI 输出 50 分
你："AI 太笨了！"
放弃不用

第 3 次：AI 输出 70 分
你："还是我自己来吧。"
彻底放弃
```

**结果：** AI 永远停留在 60 分，你永远在从头写。

---

### ✅ 正确方式（5 次迭代）

```
第 1 次：AI 输出 60 分
你："方向对了，但开头不够吸引人，换成'姐妹们！今天说个扎心的真相...'"

第 2 次：AI 输出 70 分
你："好多了！正文分 3 段，现在太长了。"

第 3 次：AI 输出 80 分
你："不错！结尾加个互动引导，让读者评论。"

第 4 次：AI 输出 85 分
你："很好！再调整一下语气，更活泼一点。"

第 5 次：AI 输出 90 分
你："完美！这个指令保存下来，下次还用。"
```

**结果：** AI 从 60 分提升到 90 分，你有了一个可用的模板。

## 💡 六、核心心态转变

### 从"AI 应该懂我"到"我应该教 AI"

❌ **错误心态：**
```
"我都说这么清楚了，它怎么还不懂？"
"这 AI 太笨了，理解不了我的意思。"
```

✅ **正确心态：**
```
"我没说清楚，它当然不懂。"
"我换个说法，它应该就能懂了。"
```

**记住：AI 不是你肚子里的蛔虫。**

你不说清楚，它真的不知道。

---

### 从"一次完美"到"逐步迭代"

❌ **错误心态：**
```
"第一次就应该完美。"
"还要我改？这不智能啊。"
```

✅ **正确心态：**
```
"第一次是草稿，正常。"
"改一改就更好了。"
```

**记住：好作品是改出来的，不是写出来的。**

---

### 从"AI 取代我"到"AI 辅助我"

❌ **错误心态：**
```
"AI 应该自己搞定，我还要它干嘛？"
"这还得我改，不如我自己写。"
```

✅ **正确心态：**
```
"AI 做初稿，我来改。"
"它做 80 分，我做到 90 分。"
```

**记住：AI 是辅助，不是取代。**

## 📝 七、课后练习

**下次用 AI 时，试试这个流程：**

1. 给清晰的任务（角色 + 任务 + 要求）
2. 给 1-2 个示例（Few-Shot）
3. AI 输出后，给具体反馈
4. 迭代 3-5 次，别放弃
5. 好了之后，保存指令模板

**坚持一周，你会发现 AI 好用 10 倍。**
