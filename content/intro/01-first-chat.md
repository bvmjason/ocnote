> 
> 别问我是怎么知道的，问就是被 AI 气过。😤

## 🤔 AI 不是搜索引擎

很多人第一次用 AI 时，会把它当 Google 用：

```
❌ "在吗？"
❌ "帮我查一下天气"
❌ "那个谁，就是写代码很厉害的那个"
```

然后 AI 就懵了：

```
🤖 "请问您具体需要查询哪个城市的天气？"
🤖 "请问您说的是哪位开发者？"
```

**AI 不是搜索引擎，它是一个需要明确指令的执行器。**

搜索引擎是你给它关键词，它给你一堆链接。
AI 是你给它指令，它直接给你结果。

这俩不是一回事。

就像你让外卖小哥"随便买点吃的"，他敢吗？他不敢。
因为他不知道你要什么，怕买错了你给差评。

AI 也一样。

## ✅ 正确的打开方式

### 1. 别问"在吗"，直接说事

**错误示范：**
```
你：在吗？
AI：在的，有什么可以帮您？
你：我想写个东西
AI：好的，请问您想写什么类型的东西呢？
你：就是那种...请假用的
AI：好的，请问是病假还是事假？需要包含哪些内容？
你：哎呀好麻烦
```

**正确姿势：**
```
你：帮我写一封请假邮件，明天要去看医生，语气正式一点
AI：好的，已为您生成...
```

看到没？后者一句话搞定，前者来来回回说了 5 句。

**时间就是金钱啊朋友。**

---

### 2. 给 AI 一个身份

**错误示范：**
```
你：写个文案
AI：好的，请问是什么产品的文案？什么风格？
```

**正确姿势：**
```
你：你是一个资深文案，帮我写一个小红书风格的咖啡店推广文案
```

为什么给身份有用？

因为不同身份的人，说话方式不一样。

- 你让"资深文案"写，它会用专业术语
- 你让"小红书博主"写，它会用 emoji 和感叹号
- 你让"科技博主"写，它会用数据和测评

**你给什么身份，AI 就演什么角色。**

就像你找装修师傅，你说"我要装修"，他肯定问你"装什么风格"。
但你说"我要装北欧风"，他直接给你出方案。

---

### 3. 说清楚你要什么

**错误示范：**
```
你：写得好点
AI：好的...（改了）
你：还是不够好
AI：...（又改了）
你：算了算了
```

**正确姿势：**
```
你：用轻松幽默的语气，200 字以内，目标受众是 25-35 岁的上班族
```

"好"是个主观词。
你觉得好，AI 不一定知道怎么个好法。

但你说"幽默"、"200 字"、"25-35 岁"，这些是客观的。
AI 知道怎么做。

**模糊的要求 = 模糊的结果。**

---

## 🎯 万能公式

<div class="formula-box">

**[角色] + [任务] + [要求] = 好指令**

- **角色**：你希望 AI 扮演什么
- **任务**：你要 AI 做什么
- **要求**：具体的限制条件

</div>

**举个例子：**

```
你是一个健身教练（角色），
帮我制定一个减脂计划（任务），
要求每周 3 次训练，每次 30 分钟，不需要器械（要求）。
```

这个公式，90% 的场景都够用。

**再举几个例子：**

```
你是一个 Python 专家（角色），
帮我写一个爬虫脚本（任务），
要求能抓取豆瓣电影 Top250，保存为 CSV（要求）。
```

```
你是一个旅行规划师（角色），
帮我规划一个日本 7 日游（任务），
要求包含东京、大阪、京都，预算 2 万人民币（要求）。
```

```
你是一个职业规划师（角色），
帮我分析转行做产品经理的可行性（任务），
要求考虑我的背景（3 年销售经验），给出具体建议（要求）。
```

看到没？都是这个套路。

**记住这个公式，你就超越了 80% 的 AI 用户。**

## 🚗 常见翻车现场

| 错误示范 | 问题在哪 | 正确姿势 |
|---------|---------|---------|
| "随便" | AI 最恨的词 | 给出具体选项或偏好 |
| "写得好点" | 太模糊 | "用更正式/幽默/简洁的语气" |
| 一次让 AI 干 10 件事 | AI 会懵 | 一次说一件事，或分步骤 |
| "你懂我意思吧" | AI 不懂 | 把"意思"说明白 |
| "就像那个谁写的" | AI 不知道"那个谁"是谁 | 说具体名字或给示例 |

**翻车最多的："随便"**

真的，别跟 AI 说"随便"。
AI 不是对象，它真的会给你"随便"的结果。

然后你一看："这什么玩意？"
AI："你说随便的啊..."

**谁痛苦？你痛苦。**

## 📝 课后练习

**把下面的"烂指令"改成"好指令"：**

1. "帮我写个东西"
2. "做个好看的 PPT"
3. "分析一下这个数据"

**参考答案：**

1. "帮我写一封感谢信给合作团队，语气真诚，300 字以内"
2. "做一个产品介绍 PPT，10 页左右，风格简洁商务，包含产品优势、定价、案例"
3. "分析这份销售数据，找出增长最快的产品和下降最严重的产品，给出可能原因"

**做完练习，你就入门了。**

## 💡 最后的忠告

1. **别怕说太多** - AI 不会嫌你啰嗦，它怕你说太少
2. **别用暗示** - AI 听不懂暗示，直接说
3. **别指望 AI 猜** - 它不是算命先生
4. **多试几次** - 第一次不满意，调整指令再来
