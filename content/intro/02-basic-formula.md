> 
> 就像做菜，有菜谱和没菜谱，做出来的味道就是不一样。
> 
> 这个公式，就是你的 AI 菜谱。📖

## 🎯 核心公式

<div class="formula-box">

**完美指令 = 角色 + 任务 + 背景 + 要求 + 格式**

别被吓到，其实很简单。我一个一个给你拆解。

</div>

**先看个完整例子：**

```
你是一个有 10 年经验的咖啡师（角色），
帮我写一篇咖啡选购指南（任务），
目标读者是刚开始喝咖啡的新手（背景），
要求用轻松幽默的语气，2000 字以内（要求），
包含咖啡豆种类、烘焙程度、冲泡方法三个部分（格式）。
```

看懂没？就是这个套路。

下面我一个个说。

---

## 1️⃣ 角色（Role）- 给 AI 一个身份

**为什么要给角色？**

因为不同身份的人，说话方式不一样。

- 你让"大学教授"讲咖啡，他会说"咖啡因的化学结构是 C8H10N4O2"
- 你让"咖啡师"讲咖啡，他会说"这豆子有坚果和巧克力的风味"
- 你让"小红书博主"讲咖啡，他会说"姐妹们！这个豆子绝绝子！"

**你给什么角色，AI 就用什么方式说话。**

### ❌ 错误示范

```
写一篇关于咖啡的文章
```

AI 懵了：我用什么身份写？学术的？科普的？还是营销的？

### ✅ 正确示范

```
你是一个有 10 年经验的咖啡师，同时也是小红书美食博主...
```

AI 明白了：哦，要用专业但接地气的方式讲。

### 📋 常用角色模板（直接抄）

```
- 你是一个资深文案专家
- 你是一个有 15 年经验的 Python 架构师
- 你是一个米其林三星餐厅的主厨
- 你是一个拥有百万粉丝的短视频创作者
- 你是一个哈佛商学院的 MBA 导师
- 你是一个资深 HR，看过 10000+ 份简历
- 你是一个旅行博主，去过 50 个国家
- 你是一个健身教练，擅长减脂增肌
```

**记住：角色越具体，AI 表现越专业。**

---

## 2️⃣ 任务（Task）- 说清楚要 AI 做什么

**这是最核心的部分。**

任务说不清楚，AI 做出来肯定不对。

### ❌ 错误示范

```
帮我弄一下这个
```

AI：？？？哪个这个？怎么弄？

### ✅ 正确示范

```
帮我写一封给客户的道歉邮件，承认我们的交付延期...
```

AI：明白了，写道歉邮件，关于交付延期。

### 📋 任务动词库（不知道怎么说时用）

```
写、分析、总结、翻译、改写、优化、对比、解释、演示、设计、
规划、整理、提取、生成、评估、建议、检查、改进...
```

**小技巧：** 实在不知道怎么说，就用"帮我..."开头。

- 帮我写...
- 帮我分析...
- 帮我总结...
- 帮我优化...

简单粗暴，但有效。

---

## 3️⃣ 背景（Context）- 给 AI 足够的信息

**为什么需要背景？**

因为 AI 不是你肚子里的蛔虫。

你不说，它真的不知道。

### 示例对比

**❌ 没背景：**
```
写一封道歉邮件
```

AI 写的可能是：道歉迟到的、道歉发错消息的、道歉忘回消息的...

**✅ 有背景：**
```
我们公司是一家初创的 SaaS 企业，主要客户是中小企业。
昨天因为服务器故障，导致客户数据丢失了 2 小时。
现在需要给受影响的客户写一封道歉邮件。
```

AI 写的就是：道歉服务器故障导致数据丢失的。

**看到了吗？有天壤之别。**

### 📋 背景信息包括什么

```
- 你是谁（个人/公司/组织）
- 你的目标受众是谁
- 发生了什么事
- 为什么要做这个
- 有什么限制条件
```

**记住：背景越充分，AI 理解越准确。**

---

## 4️⃣ 要求（Requirements）- 告诉 AI 你的标准

**要求就是"怎么做"。**

### 常见要求类型

```
语气要求：
- 用轻松幽默的语气
- 用正式专业的语气
- 用亲切友好的语气

长度要求：
- 200 字以内
- 1000 字左右
- 不超过 5 页

内容要求：
- 包含 3 个具体案例
- 不要使用专业术语
- 重点突出性价比

风格要求：
- 像小红书风格
- 像科技媒体风格
- 像朋友聊天一样
```

### ❌ 模糊要求

```
写得好点
```

AI：什么叫"好点"？

### ✅ 具体要求

```
用轻松幽默的语气，2000 字以内，不要使用专业术语
```

AI：明白了。

**记住：要求越具体，结果越可控。**

---

## 5️⃣ 格式（Format）- 告诉 AI 输出什么样

**格式就是"长什么样"。**

### 常见格式

```
- Markdown 格式
- 表格形式
- 列表形式
- 邮件格式
- PPT 大纲
- JSON 格式
- 代码格式
```

### 示例

```
用 Markdown 格式，包含标题、小标题、列表

输出为表格，包含以下列：产品名称、价格、优缺点、推荐指数

用邮件格式，包含主题、称呼、正文、署名
```

**记住：格式越明确，后期处理越省事。**

---

## 🎯 完整示例（从烂到好）

### 烂指令
```
写个文案
```

### 普通指令
```
帮我写一个咖啡店的推广文案
```

### 好指令
```
你是一个资深文案专家（角色），
帮我写一个咖啡店的推广文案（任务），
我们是一家新开的手冲咖啡店，目标客户是 25-35 岁的白领（背景），
要求用轻松文艺的语气，500 字以内，突出"慢生活"理念（要求），
包含标题、正文、行动号召三个部分（格式）。
```

**看到差距了吗？**

烂指令：AI 不知道写什么，随便写
普通指令：AI 知道写咖啡，但不知道怎么写
好指令：AI 知道写什么、怎么写、写成什么样

**结果能一样吗？**

## 📝 课后练习

**把下面的"烂指令"改成"好指令"：**

1. "帮我做个 PPT"
2. "分析一下这个数据"
3. "写个方案"

**参考答案：**

1. "你是一个资深设计师（角色），帮我做一个产品介绍 PPT（任务），我们是一家 SaaS 公司，目标客户是中小企业（背景），要求 10 页左右，风格简洁商务，包含产品优势、定价、案例（要求），用 PPT 大纲格式（格式）。"

2. "你是一个数据分析师（角色），帮我分析这份销售数据（任务），我们是电商公司，这是 Q1 的销售数据（背景），要求找出增长最快的产品和下降最严重的产品，给出可能原因（要求），用表格 + 文字分析格式（格式）。"

3. "你是一个营销专家（角色），帮我写一个双 11 营销方案（任务），我们是一家美妆品牌，目标客户是 18-30 岁女性（背景），要求预算 50 万以内，包含社交媒体、KOL 合作、优惠活动（要求），用方案文档格式（格式）。"

**做完练习，这个公式你就掌握了。**

## 💡 最后的忠告

1. **别一次记住所有** - 先用"角色 + 任务"，熟练了再加其他
2. **别追求完美** - 第一次不满意，调整再来
3. **多抄多练** - 看到好的指令，保存下来模仿
4. **建立自己的模板** - 常用的指令，保存为模板
