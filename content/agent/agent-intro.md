# Agent 列表 教学专区

**打造你的专属 AI 团队**

---

## 什么是 AI Agent？

AI Agent 不是通用的"AI 助手"，而是**具有专业身份、独特个性和明确交付目标的专家**。

### Agent vs 普通 AI

| 特性 | 普通 AI | AI Agent |
|------|--------|---------|
| **身份** | 通用助手 | 领域专家（如"前端架构师"） |
| **目标** | 回答问题 | 交付具体成果（代码、方案、报告） |
| **风格** | 中性客观 | 独特个性和沟通方式 |
| **输出** | 通用建议 | 可执行的工作流程和代码 |

---

## 为什么需要 AI Agent？

**场景 1：开发新功能**

❌ 普通 AI：
```
帮我写一个登录页面
```
→ 得到通用代码，可能需要多次修改

✅ AI Agent：
```
你是一个资深前端工程师，请帮我写一个登录页面
要求：React + TailwindCSS，包含表单验证，有错误提示
```
→ 得到专业代码，包含最佳实践和完整测试

**场景 2：市场分析**

❌ 普通 AI：
```
分析一下市场规模
```
→ 得到泛泛而谈的结论

✅ AI Agent：
```
你是一个市场分析师，请帮我分析中国 SaaS 市场
要求：TAM/SAM/SOM 测算，竞品分析，进入策略建议
```
→ 得到结构化报告，有数据支撑和可执行建议

**场景 3：数据采集**

❌ 普通 AI：
```
帮我爬取这个网站的数据
```
→ 可能违反 robots.txt，没有异常处理

✅ AI Agent：
```
你是一个数据采集专家，请帮我采集这个网站的数据
要求：遵守 robots.txt，控制请求频率，有异常处理机制
```
→ 得到合规、稳定、可维护的爬虫代码

## 快速开始

### 步骤 1：选择你的 Agent

根据你的需求选择合适的 Agent：
- **开发新功能** → 前端工程师 + 后端架构师
- **性能优化** → DevOps 自动化专家 + 数据库优化师
- **安全审计** → 安全工程师 + 代码审查师
- **快速验证** → 快速原型师
- **市场调研** → 市场分析师
- **数据采集** → 数据采集专家
- **文案撰写** → 文案撰写专家
- **方案定制** → 方案定制专家
- **流程优化** → 流程管理专家

### 步骤 2：激活 Agent

```markdown
你是一个 [Agent 名称]，请帮我 [具体任务]

【项目背景】
（描述你的项目）

【具体要求】
（列出具体需求）

【交付物】
（期望的输出形式）
```

### 步骤 3：获取交付物

每个 Agent 都会提供：
- 专业的工作流程
- 可执行的代码/方案
- 成功指标和验证方法

## 使用技巧

### 1. 明确 Agent 身份

**❌ 错误：**
```
帮我写一个登录页面
```

**✅ 正确：**
```
你是一个资深前端工程师，请帮我写一个登录页面
要求：React + TailwindCSS，包含表单验证
```

### 2. 提供上下文

**❌ 错误：**
```
优化这个函数
```

**✅ 正确：**
```
你是一个性能优化专家，请优化这个函数
背景：这是电商网站的商品列表渲染函数，目前有卡顿
目标：首屏加载时间 < 1 秒
```

### 3. 指定交付物

**❌ 错误：**
```
怎么做用户认证？
```

**✅ 正确：**
```
你是一个安全工程师，请设计用户认证方案
交付物：
1. 技术选型对比表
2. JWT 实现代码
3. 安全 Checklist
```
