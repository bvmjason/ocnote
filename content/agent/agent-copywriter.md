## 核心技能

### 1. 营销文案

**适用场景：**
- 产品发布文案
- 广告创意
- 落地页文案
- 邮件营销

**指令模板：**
```markdown
你是一个文案专家，请帮我写 [产品类型] 的营销文案

【产品信息】
- 产品名称：[名称]
- 核心功能：[3-5 个功能点]
- 目标用户：[用户画像]
- 价格：[价格区间]

【文案要求】
- 调性：[专业/活泼/高端/亲民]
- 长度：[字数要求]
- 渠道：[微信/抖音/官网/邮件]
- CTA: [期望的行动]

【交付物】
1. 主标题（5 个版本）
2. 副标题（3 个版本）
3. 正文文案
4. CTA 文案
5. 社交媒体版本（适配各平台）
```

**输出示例：**
```markdown
# 产品营销文案

## 主标题（5 个版本）
1. [痛点型] 还在为 [痛点] 烦恼？这个产品帮你解决
2. [利益型] 每天 10 分钟，[收益] 翻 3 倍
3. [好奇型] 为什么 90% 的 [目标用户] 都在用它？
4. [数字型] 3 个步骤，7 天见效，100% 满意
5. [对比型] 比 [竞品] 便宜 50%，效果却好 2 倍

## 副标题（3 个版本）
1. [功能说明] 核心功能 + 用户收益
2. [信任背书] 用户数/获奖/认证
3. [限时优惠] 促销活动信息

## 正文文案
【开头 - 引起共鸣】
你是不是也遇到过这些问题...

【中间 - 产品介绍】
我们的产品可以帮你...

【结尾 - 行动号召】
现在购买，立享优惠...

## CTA 文案
- 立即购买
- 免费试用
- 立即咨询
- 限时优惠

## 社交媒体版本
### 微信（长文案）
[完整版]

### 微博（短文案）
[140 字版本]

### 抖音（脚本）
[视频脚本]
```

## 成功案例

### 案例 1：SaaS 产品落地页文案

**背景：**
- B 端 SaaS 产品
- 转化率低（2%）
- 需要提升转化

**Agent 方案：**
1. 用户痛点分析
2. 价值主张重构
3. 社会证明强化
4. CTA 优化

**结果：**
- 转化率：2% → 5.5%
- 停留时间：+40%
- 咨询量：+150%

4. 多渠道传播

**结果：**
- 品牌认知度：+200%
