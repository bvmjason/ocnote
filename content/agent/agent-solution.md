## 核心技能

### 1. 解决方案设计

**适用场景：**
- 企业数字化转型
- 业务流程优化
- 技术选型方案

**指令模板：**
```markdown
你是一个方案专家，请帮我设计 [项目/问题] 的解决方案

【背景】
- 公司/项目：[介绍]
- 核心问题：[描述问题]
- 目标：[期望结果]
- 约束：[预算/时间/资源]

【要求】
- 方案完整可执行
- 有成功案例参考
- 包含成本估算
- 风险评估

【交付物】
1. 现状分析
2. 解决方案设计
3. 实施计划
4. 资源配置
5. 成本估算
6. 风险评估
7. 成功指标
```

**输出框架：**
```markdown

- 机会 1：[描述]
```

## 成功案例

### 案例 1：企业数字化转型方案

**背景：**
- 传统制造企业
- 信息化程度低
- 需要数字化转型

**Agent 方案：**
1. 现状调研（2 周）
2. 方案设计（3 周）
3. 试点实施（8 周）
4. 全面推广（16 周）

**结果：**
- 生产效率：+35%
- 库存周转：+50%
- 客户满意度：+25%

### 案例 2：营销活动执行方案

**背景：**
- 双 11 营销活动
- 多平台同步
- 预算 500 万

**Agent 方案：**
1. 活动方案设计
2. 执行 SOP
3. 应急预案
4. 数据监控

**结果：**
- GMV: 3000 万
- ROI: 1:6
- 零重大事故
