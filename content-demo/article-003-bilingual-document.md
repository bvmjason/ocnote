## 🛠️ 实战演示

### 案例：产品规格书中英文对照

**中文原文：**
```
BM32F103 是一款 32 位 ARM Cortex-M3 微控制器，
主频高达 72MHz，内置 64KB Flash 和 20KB SRAM。

主要特性：
• 32 位 ARM Cortex-M3 内核
• 最高工作频率 72MHz
• 64KB 闪存，20KB 静态存储器
• 丰富的外设接口
• 工作温度范围 -40°C ~ +85°C
```

**AI 翻译英文：**
```
The BM32F103 is a 32-bit ARM Cortex-M3 microcontroller
with a maximum frequency of 72MHz, featuring 64KB Flash
and 20KB SRAM.

Key Features:
• 32-bit ARM Cortex-M3 core
• Maximum operating frequency 72MHz
• 64KB Flash memory, 20KB SRAM
• Rich peripheral interfaces
• Operating temperature range -40°C ~ +85°C
```

### 专业术语处理

**电子产业专业术语：**
```
中文 → 英文（AI 翻译）

微控制器 → Microcontroller (MCU)
闪存 → Flash Memory
静态存储器 → Static RAM (SRAM)
外设接口 → Peripheral Interfaces
工作温度 → Operating Temperature
封装 → Package
应用电路 → Application Circuit
电气参数 → Electrical Characteristics
```

**AI 的优势：**
```
✓ 术语准确（基于电子产业词库）
✓ 上下文理解（知道 MCU 是什么）
✓ 格式保持（表格/列表不变）
✓ 一致性（同一术语全文统一）
```
